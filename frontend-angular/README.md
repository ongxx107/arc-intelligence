# Smart Support Inbox (Django REST Frontend)

Frontend for the **Smart Support Inbox** coding challenge: store inbound support tickets, generate deterministic analysis (category, priority, summary, suggested reply), and allow agents to submit feedback (accept/reject).  

## Setup (local)

```bash
npm install
```

This will create the `node_modules` directory and install all required packages.

## Run the Angular Dev Server

```bash
npm start
```

Once compiled, open:

```
http://localhost:4200/
```
